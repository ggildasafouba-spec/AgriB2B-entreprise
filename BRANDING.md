# Branding AgroMarket

## Logo professionnel

### Concept visuel
- Un symbole simple et moderne combinant une feuille stylisée et un panier de marché.
- Le symbole doit rester lisible en petit format et facilement reconnaissable.
- Variante graphique : une feuille verte formant le contour d'un panier, avec un petit chemin ou une goutte pour évoquer la croissance et la fraîcheur.

### Wordmark
- Police sans-serif moderne, ronde et élégante.
- Exemple de typographie : `Inter`, `Poppins` ou `Montserrat`.
- Le mot "AgroMarket" peut se présenter en deux tons : "Agro" en vert, "Market" en gris foncé.

### Logo texte
- Version texte : `AgroMarket`
- Version courte : `AM` avec un symbole de feuille/panier.

## Slogan simple et mémorable

> **"AgroMarket — La terre vendue en toute confiance."**

Autres variantes :
- « Connecter les fermes au marché, simplement. »
- « Le marché agricole de confiance. »
- « Du champ à vous, en toute transparence. »

## Palette de couleurs cohérente

1. **Vert principal**
   - Code : `#2E7D32`
   - Usage : identité, boutons principaux, éléments d'action.

2. **Vert clair**
   - Code : `#A5D6A7`
   - Usage : arrière-plans doux, bordures, statuts positifs.

3. **Or / jaune chaud**
   - Code : `#F9A825`
   - Usage : accents, appels à l'action secondaires, badges.

4. **Gris foncé**
   - Code : `#263238`
   - Usage : texte principal, titres, éléments sérieux.

5. **Gris neutre**
   - Code : `#ECEFF1`
   - Usage : arrière-plan, cartes, interfaces.

6. **Blanc**
   - Code : `#FFFFFF`
   - Usage : espaces, contraste, lisibilité.

## Vision claire

### Vision d'AgroMarket
- Faire d'AgroMarket la plateforme agricole de référence pour les producteurs et les acheteurs urbains.
- Créer un écosystème numérique où la qualité, la transparence et le paiement sécurisé sont au cœur de chaque transaction.
- Réduire les intermédiaires et garantir un revenu juste aux agriculteurs.
- Permettre aux consommateurs d'accéder à des produits frais, traçables et abordables.

### Positionnement
- **AgroMarket est la réponse locale à un marché agricole digitalisé.**
- La plateforme est conçue pour être simple, fiable et utile aux acteurs du terrain.
- Son objectif est de renforcer la confiance entre agriculteurs et acheteurs tout en valorisant la production locale.

## Utilisation recommandée

- Mettre le logo en haut à gauche de l'application.
- Placer le slogan sous le logo ou sur la page d'accueil.
- Utiliser le vert principal pour les actions importantes, et l'or pour les éléments qui doivent attirer l'attention.
- Garder l'interface claire, avec des blocs blancs et des contrastes doux pour mettre en valeur les informations essentielles.

## Exemple de wording pour l'application
- Page d'accueil : "Achetez local. Vendez mieux."
- Section vendeur : "Augmentez vos revenus, sans intermédiaire."
- Section acheteur : "Des produits frais, livrés en toute confiance."
- Page de paiement : "Paiement sécurisé, dès que la transaction est confirmée."
