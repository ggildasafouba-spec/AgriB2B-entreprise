## 1. QU'EST-CE QU'AGRIB2B ?

AgriB2B est une plateforme numérique qui met en relation les producteurs agricoles, les acheteurs et les transporteurs au Cameroun. Elle permet de :
- Vendre ses produits agricoles en ligne
- Acheter directement auprès des producteurs
- Proposer des services de transport
- Sécuriser les paiements entre les parties

## 2. COMMENT S'INSCRIRE ?

1. Ouvrir le lien : https://agri-b2-b-entreprise.vercel.app
2. Cliquer sur "Inscription"
3. Choisir son type de compte :
   - Acheteur particulier ou entreprise
   - Producteur particulier ou entreprise
   - Transporteur particulier ou entreprise
4. Remplir le formulaire (nom, email, téléphone, pays, région)
5. Cocher "J'accepte les conditions générales d'utilisation"
6. Cliquer "Créer mon compte"
7. Le compte est immédiatement actif

**Note :** Les entreprises doivent fournir un document justificatif (registre de commerce, statuts, etc.)

## 3. COMMENT SE CONNECTER ?

## 4. LES 3 MODES D'UTILISATION

Après connexion, l'utilisateur voit 3 options :

- **Acheter** — Pour parcourir les produits et passer des commandes
- **Vendre** — Pour publier ses produits et gérer ses ventes
- **Transporter** — Pour proposer des tarifs de transport et gérer des livraisons

**Tous les utilisateurs peuvent utiliser les 3 modes.** Un producteur peut aussi acheter des produits ou proposer du transport. Il suffit de cliquer sur le mode souhaité.

---

## 5. MODE ACHETEUR

### Passer une commande
1. Aller dans "Commandes"
2. Cliquer "Nouvelle commande"
3. Sélectionner les produits souhaités (cliquer "+ Ajouter")
4. Vérifier le récapitulatif (total, commission, montant vendeur)
5. Cliquer "Confirmer la commande"

### Suivre sa commande
La commande passe par ces étapes :
- **En attente** — Le vendeur n'a pas encore répondu
- **Confirmée** — Le vendeur a accepté et prépare le colis
- **Expédiée** — Le colis est en route
- **Livrée** — Vous avez confirmé la réception

### Confirmer la réception
Quand le vendeur marque la commande comme "Expédiée", un bouton apparaît :
**"J'ai bien reçu ma commande"**
En cliquant dessus, le paiement est libéré au vendeur.

### Annuler une commande
Tant que le vendeur n'a pas accepté, l'acheteur peut cliquer :
**"Annuler ma commande"**

### Commander une livraison
Après avoir passé une commande, l'acheteur peut :
1. Cliquer "Commander une livraison"
2. Choisir le type de service (Express, Standard, Économique, Frigorifique)
3. Sélectionner un transporteur et un trajet
4. Entrer l'adresse de livraison
5. Confirmer

---

## 6. MODE VENDEUR

### Publier un produit
1. Aller dans "Produits"
2. Cliquer "Nouveau produit"
3. Remplir : nom, description, prix (FCFA/kg), catégorie, zone de production
4. Indiquer le stock disponible
5. Sélectionner les moyens de transport disponibles
6. Cliquer "Publier le produit"

### Modifier un produit
1. Survoler le produit
2. Cliquer l'icône bleue (crayon)
3. Modifier les informations
4. Cliquer "Enregistrer les modifications"

### Supprimer un produit
1. Survoler le produit
2. Cliquer l'icône rouge (poubelle)
3. Confirmer la suppression

### Gérer les commandes reçues
Quand un acheteur passe une commande :
1. Le vendeur reçoit une notification
2. Il va dans "Commandes"
3. Il voit le bouton **"Accepter la commande"** ou **"Refuser"**
4. Après acceptation, il prépare le colis
5. Il clique **"Colis expédié"** quand c'est envoyé
6. Il attend que l'acheteur confirme la réception
7. Le paiement est libéré (moins la commission)

### Gérer le stock
- Aller dans "Stock"
- Le stock se met à jour automatiquement après chaque commande
- Possibilité de modifier manuellement les quantités

---

## 7. MODE TRANSPORTEUR

### Proposer un tarif de transport
1. Aller dans "Transport"
2. Cliquer "+ Nouveau tarif"
3. Remplir :
   - Origine (ex: Douala)
   - Destination (ex: Yaoundé)
   - Catégorie de produit (ex: Légumes)
   - Prix par kg (en FCFA)
   - Type de véhicule (ex: Camion 10T)
   - Délai estimé (en jours)
4. Cliquer "Créer le tarif"

**Note :** Une commission de 3% est automatiquement ajoutée au prix affiché aux clients. Le transporteur reçoit son prix de base.

### Gérer ses livraisons
Quand un acheteur commande une livraison :
1. Le transporteur reçoit une notification
2. Il va dans la page de suivi de livraison
3. Il met à jour le statut étape par étape :
   - Accepter la livraison
   - Colis récupéré
   - En transit
   - Arrivé en ville
   - En cours de livraison
   - Livré

## 8. LE SYSTÈME DE PAIEMENT (ESCROW)

### Comment ça marche ?
L'escrow (séquestre) protège l'acheteur ET le vendeur :

1. L'acheteur paie → l'argent est **bloqué** sur la plateforme
2. Le vendeur expédie → l'acheteur confirme la réception
3. L'argent est **libéré** au vendeur (moins la commission)

### En cas de problème :
- Si le vendeur ne livre pas → l'acheteur est remboursé
- Si l'acheteur ne confirme pas → l'admin peut intervenir
- Si la commande est annulée → remboursement automatique

### Modes de paiement acceptés :
- MTN Mobile Money
- Orange Money
- Paiement manuel (virement, confirmé par l'admin)

## 9. LES COMMISSIONS

| Qui | Commission | Exemple |
|-----|-----------|---------|
| Vendeur particulier | 5% | Vente de 100 000 FCFA → il reçoit 95 000 FCFA |
| Vendeur entreprise | 10% | Vente de 100 000 FCFA → il reçoit 90 000 FCFA |
| Transporteur | 3% | Tarif de 50 FCFA/kg → client paie 51,5 FCFA/kg |

La commission est prélevée automatiquement. Le vendeur/transporteur reçoit le montant net.

## 12. LE SUIVI DE LIVRAISON

### Les étapes de livraison :
1. **En attente** — Livraison créée, en attente du transporteur
2. **Acceptée** — Le transporteur a pris en charge
3. **Récupéré** — Le colis est récupéré chez le vendeur
4. **En transit** — Le colis est en route
5. **Arrivé en ville** — Le colis est dans la ville de destination
6. **En livraison** — Le livreur est en route vers l'adresse
7. **Livré** — Le colis est remis au destinataire

### Types de service :
- **Express** (1-2 jours) — Pour les produits très périssables (tomates, laitue)
- **Standard** (2-4 jours) — Pour la majorité des produits
- **Économique** (4-7 jours) — Pour les produits non périssables (céréales, épices)
- **Frigorifique** (1-3 jours) — Pour les produits sensibles à la chaleur (viande, poisson)

## 16. QUESTIONS FRÉQUENTES (FAQ)

**Q : L'inscription est-elle gratuite ?**
R : Oui, l'inscription est totalement gratuite.

**Q : Puis-je être vendeur ET acheteur ?**
R : Oui, tous les utilisateurs peuvent acheter, vendre et transporter. Il suffit de choisir le mode souhaité après connexion.

**Q : Comment suis-je payé en tant que vendeur ?**
R : Quand l'acheteur confirme la réception de sa commande, le paiement est libéré sur votre compte Mobile Money (moins la commission).

**Q : Que se passe-t-il si l'acheteur ne paie pas ?**
R : Le système d'escrow bloque l'argent avant l'expédition. Le vendeur n'expédie que quand le paiement est confirmé.

**Q : Que se passe-t-il si le vendeur ne livre pas ?**
R : L'argent reste bloqué dans l'escrow. Si le vendeur ne livre pas, l'acheteur est remboursé.

**Q : Pourquoi je ne peux pas envoyer mon numéro de téléphone par message ?**
R : Pour votre sécurité. Toutes les transactions doivent passer par la plateforme pour bénéficier de la protection escrow.

**Q : La plateforme garantit-elle la qualité des produits ?**
R : Non. AgriB2B est une plateforme de mise en relation. La qualité des produits est sous la responsabilité du vendeur. Les avis et notes permettent d'identifier les vendeurs fiables.

**Q : La plateforme gère-t-elle les stocks ?**
R : Non. La gestion des stocks est déclarative — c'est le vendeur qui indique ses quantités disponibles. AgriB2B ne vérifie pas les stocks réels.

**Q : Comment supprimer mon compte ?**
R : Contactez l'administrateur à contact.agrib2b@gmail.com ou demandez la suppression via la messagerie.

**Q : Combien coûte l'utilisation de la plateforme ?**
R : L'utilisation est gratuite. Seule une commission est prélevée sur les transactions réalisées (5% particuliers, 10% entreprises, 3% transport).
